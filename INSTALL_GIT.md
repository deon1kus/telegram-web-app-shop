# Установка Git для завершения коммита

Git не найден в системе. Для завершения сохранения версии v001 нужно установить Git.

## Быстрая установка Git

### Вариант 1: Скачать установщик

1. Перейдите на https://git-scm.com/download/win
2. Скачайте установщик (Git for Windows)
3. Запустите установщик и следуйте инструкциям
4. После установки перезапустите PowerShell
5. Выполните команды из `GIT_COMMANDS.md`

### Вариант 2: Использовать winget (Windows 10/11)

```powershell
winget install --id Git.Git -e --source winget
```

### Вариант 3: Использовать Chocolatey

```powershell
choco install git
```

## После установки Git

Выполните команды:

```bash
cd D:\WebApp\telegram-web-app-shop-local
git add .
git commit -m "feat: рабочая версия v001 - полностью переведена на русский, исправлены все ошибки"
git tag v001
```

## Альтернатива: Выполнить на сервере

Если не хотите устанавливать Git локально, можно выполнить все команды на сервере после копирования файлов туда.
